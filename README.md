# OS Assistant - ассистент операционной системы Windows

## Описание

Программа разработана для поиска и запуска приложений Windows по описанию задачи пользователя.

Интеллектуальным агентом в текущей версии выступает модель **GigaChat**, эмбеддинг модель - **rubert-tiny2**.
 
Доступен ***режиме отладки*** с выводом отладочной информации в файл лога.

## Использование

Для запуска **OS Assistant** используйте команду: ```amd64/python main.py```

### Режим отладки

Для работы в ***режиме отладки*** используйте команду: ```amd64/python main.py debug```.

В данном режиме в файл лога выводится информация, которой обмениваются агенты **OS Assistant**.

## Установка

Порядок установки:
1. Создайте новый каталог **OS Assistant**;
2. Скачайте репозиторий программы, для этого:
	- Если у вас установлен Git:
		- В командной строке перейдите в каталог : ```cd <путь к папке OS Assistant>```;
		- Выполните команду клонирования репозитория: ```git clone https://github.com/sskondakov/RosNOU_Pr4.git```
	- Если Git не установлен:
		- На странице https://github.com/sskondakov/RosNOU_Pr4 в меню списка репозитория выберите команду *Code->Download ZIP*
		- Распакуйте скачанный архив в каталог **OS Assistant**
3. В командной строке перейдите в каталог **OS Assistant**: ```cd <путь к папке OS Assistant>```;
4. Выполните команду: ```amd64/python install.py```
5. В случае успешной установки вы должны увидеть сообщение: ```Программа успешно установлена.```
6. Укажите ключ авторизации **GigaChat** в файле *gigakeys.ini*.

### Настройка брандмауэра
Работа **OS Assistant** может блокироваться **брандмауэром** вашего компьютера, при необходимости настройте (см. в документации **брандмауэра**):
- доступ **OS Assistant** к **GigaChat**;

## Тестирование

Для первоначального тестирования достаточно произвести установку **OS Assistant** и указать ключ авторизации **GigaChat**.

Порядок тестирования:
1. В командной строке перейдите в каталог **OS Assistant**: ```cd <путь к папке OS Assistant>```;
2. Запустите **OS Assistant**: ```amd64/python main.py```;
3. Дождитесь загрузке приложения, должна появится иконка ***OS AI*** в системной трее.
4. По правой клавише мыши вызовите контекстное меню иконки и выберите необходимый пункт:
	- ***Открыть диалог*** - В данном режиме работы вы можете кратко описать свою задачу, **OS Assistant** найдет и запустит подходящую программу
	- ***Редактор функций*** - В редакторе вы можете редактировать список функций - приложений которые "знает" **OS Assistant**
	- ***Выход*** - выход из приложения

## Настройка

Для боевого применения достаточно установить ключ авторизации **GigaChat**.

Настройка параметров **OS Assistant** выполняется в конфигурационных файлах:
1. *gigakeys.ini*:
	- секция ***GIGACHAT***
		- **authorization_key** - ключ авторизации **GigaChat** (см. в документации **сервиса**);
		- **session_id** - идентификатор сессии **GigaChat** (см. в документации **сервиса**);
2. *config.ini*:
	- секция ***FUNCTIONS_DB***:
		- **db_name** - имя файла базы данных **OS Assistant**
	- секция ***GIGACHAT***:
		- **max_context_length** - размер контекста **GigaChat** (см. в документации **сервиса**);
		- **model** - используемая модель **GigaChat** (см. в документации **сервиса**);
	- секция ***DIALOG_HISTORY***:
		- **file_name** - имя файла истории диалогов **OS Assistant**
	- секция ***RUBERT_TINY2***:
		- **folder_name** - имя папки с моделью **rubert-tiny2**
